# KINETIC.FOLIO

A Swiss brutalist portfolio showcasing animated webpage experiments using GSAP and Motion technologies.

## Features

- **Kinetic Card Grid** - Floating cards with mouse-reactive physics
- **GSAP Animations** - Staggered entrance, idle floating, hover effects
- **Iframe Viewer** - Embedded page viewing with persistent navigation
- **Swiss Brutalist Design** - Helvetica, geometric shapes, red accent (#E63946)

## Pages Included

| Page | Description |
|------|-------------|
| **KINETIC.LAB** | AI-Powered Animation Generator (Claude + GSAP) |
| **CALDER MOBILE** | Physics-based pendulum sculpture simulation |
| **ANTIGRAVITY: FLUX** | Fluid dynamics particle art with SVG goo effects |
| **GIS MASTER** | Swiss brutalist scrollytelling with canvas physics |

## Tech Stack

- **GSAP 3.12** - Animation library
- **ScrollTrigger** - Scroll-driven animations
- **Canvas 2D** - Physics simulations
- **Vanilla JavaScript** - No framework dependencies

## Local Development

```bash
# Start local server
python3 -m http.server 8888

# Open in browser
open http://localhost:8888
```
